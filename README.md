# 🛒 ShopEase - Full Stack E-Commerce Website


A beginner-level full stack e-commerce web application built to understand and implement real-world web development concepts.

This project was created as a first-year B.Tech summer project to practice frontend, backend, database, authentication, and API development.







## 🚀 Features


### 👤 User Features

- User registration and login
- JWT based authentication
- User profile management
- Browse products
- View product details
- Add products to cart
- Update cart quantity
- Remove items from cart




### 🛍️ Product Features

- Product listing
- Product details page
- Product categories
- Product ratings
- Admin product management




### 🔐 Admin Features

- Admin authentication
- Add products
- Update products
- Delete products







## 🛠️ Tech Stack


### Frontend

- HTML5
- CSS3
- JavaScript




### Backend

- Node.js
- Express.js




### Database

- MongoDB
- Mongoose




### Authentication

- JSON Web Token (JWT)
