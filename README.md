# 🛒 ShopEase

ShopEase is my **first Full Stack E-Commerce Web Application**, built from scratch to learn how modern web applications work from end to end.

The project combines a responsive frontend with a secure backend, user authentication, REST APIs, and a cloud-hosted MongoDB database. It has been an incredible learning experience that helped me understand how frontend, backend, and databases communicate with each other.

---

## 🚀 Features

- User Registration & Login
- JWT Authentication
- Secure Password Hashing using bcrypt.js
- Product Listing
- Product Details
- Shopping Cart
- User Profile
- RESTful APIs
- MongoDB Atlas Cloud Database
- Responsive UI
- Environment Variables using dotenv

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/ShopEase.git
```

### Navigate to the project

```bash
cd ShopEase
```

### Install Backend Dependencies

```bash
cd server
npm install
```

### Install Frontend Dependencies (if applicable)

```bash
cd ../client
npm install
```

### Configure Environment Variables

Create a `.env` file inside the `server` folder.

Example:

```env
PORT=5000

MONGO_URI=Your_MongoDB_Atlas_URI

JWT_SECRET=Your_JWT_Secret
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

Open the frontend or run your frontend development server if applicable.
